# Security Policy

## Supported Versions

The following versions of **WikiCrawler** are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please **do not** open a public issue. Instead, please report it through one of the following methods:

1. **GitHub Private Reporting**: Use the "Report a vulnerability" button under the **Security** tab of this repository.
2. **Email**: Contact the maintainer at [your-email@example.com] (Please replace this with your actual email or use GitHub's private reporting).

We take security seriously and will respond to your report as soon as possible.

## Disclosure Process

1. We will acknowledge receipt of your report within 48 hours.
2. We will provide a timeline for a fix and keep you updated on our progress.
3. Once a fix is ready, we will coordinate a public disclosure date with you.
